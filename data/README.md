# Data Directory

Files generated by `split_data.py`:

- `nutribench_v2_cot.csv` – raw export of the CoT-enhanced NutriBench dataset.
- `train.csv` – training split used for prompt optimization.
- `val.csv` – validation split with at least 1,000 samples.
- `metadata.json` – provenance information (seed, sizes, summary statistics).

Re-generate the splits with:

```bash
python split_data.py --output-dir data --val-size 1000 --seed 42 --overwrite
```

> Keep raw and split files out of version control (already covered by `.gitignore`).
